# Instructions for the Sinatralab GitHub

## How to join:
1. Get a GitHub account
2. Become a member of the Sinatralab GitHub organization (ask [name] or [name] to add you)
3. Each project in the lab has a separate GitHub repository

## What the project repository typically contains:
* Brief project description
* Project team + contact(s)
* Code (if any)
* Documentation (if any)
* Data (if publicly shareable) or instructions to collaborators on where to find data (if not publicly shareable)
* Instructions on where to find all other project-related material: presentations, academic articles, notes, literature etc.

The minimum working example (MWE) is a markdown file that contains the above information. To get started, you can use the templates:
* [MWE template](./mwe-template.md)
* Coding project template
